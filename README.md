# 📊 Mutual Fund Investment Plan using Python

## 📖 Project Overview
This project focuses on building a Python-based application to create and evaluate mutual fund investment plans. The goal is to help investors understand how different investment strategies perform over time by analyzing historical data and key financial metrics.

The project uses financial data such as fund prices, returns, and expense ratios to simulate portfolio performance. It also helps in understanding risk and return, making it easier to take better investment decisions.

---

## 🎯 Objective
- To analyze mutual fund performance using historical data  
- To calculate key financial metrics like returns and risk  
- To build and simulate investment portfolios  
- To provide insights for better investment decisions  

---

## 🛠️ Tools & Libraries
- Python  
- Pandas  
- NumPy  
- Matplotlib / Seaborn / Plotly  

---

## 📊 Data Collection & Preparation
- Collected historical mutual fund data (NAV, returns, expense ratio)  
- Cleaned and handled missing or inconsistent values  
- Standardized time intervals (monthly / yearly)  
- Created calculated metrics such as:
  - Total Return  
  - Annualized Return  
  - Volatility (Risk)  

---

## 🔄 Portfolio Construction
- Created portfolios with different fund allocations  
- Implemented diversification logic  
- Simulated portfolio performance based on historical data  

---

## 📈 Risk & Return Analysis
- Calculated expected portfolio returns  
- Measured risk using standard deviation  
- Evaluated performance using:
  - Sharpe Ratio  
  - Sortino Ratio  

- Applied Monte Carlo Simulation to estimate future outcomes  

---

## 📊 Visualization
- Line charts for portfolio performance over time  
- Risk vs Return scatter plots  
- Pie charts for portfolio allocation  
- Heatmaps for fund performance comparison
